## Техническая проблема

### Конфликт требований

**Для nested with `__nested__`**: нужна балансировка скобок
```
@[hello @[world]] → должен балансировать → берет второй ]
```

**Для greedy without `__nested__`**: нужен максимальный захват
```
@[hello #[world]] → должен брать последний ] → берет второй ]
```

**Но**: Эти требования конфликтуют с adjacent marks
```
**bold1** **bold2** → greedy "last" сломает первый mark
```

### Решение требует

1. **Context-aware matching** - понимать, какие паттерны могут быть внутри
2. **Multi-pass processing** - сначала inner, потом outer
3. **Complex priority system** - учитывать множество факторов

**Сложность**: Высокая (7-8/10)
**Время**: 5-8 часов дополнительно
